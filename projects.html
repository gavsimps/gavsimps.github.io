Pediatric Therapy Solutions (PTS) is a joint company between Tiny Tots Speech Therapy and PT for Kids, Inc. to encompass physical, speech, and occupational therapy in a single setting. They had reached out to me and a small group of graduates from IU to recreate their website from scratch.

We used the Flask framework to build the base of this web project. It was written using Python, HTML, CSS, and Javascript with a connected SQL database. The site allows for employee specific logins, resume management, client screenings, and automated emailing. This site is still under development, but currently fulfills all needed functions.

This website was created to demonstrate the culmination of my skills in web development at IU. I had a lot of fun creating this project as there were no constraints as to what one could make. I created mostly infomational pages documenting thnigs I enjoy or believe otherr people should know.

Some functionality of the site is deprecated as it is using links no longer available as it is hosted on IU servers. This website utilizes HTML, CSS, and Javascript to function. WARNING: On the main page there is a light blue button that plays a VERY loud sound and displays a flash effect when clicked.

I have successfully deployed and configured an external Ubuntu server as a personal project to deepen my understanding of Linux systems and server administration. I have installed and managed core services such as OpenSSH, Apache/Nginx, and UFW for firewall management.

Additionally, using systemd for service management and shell scripts to automate routine tasks, this project helped in my understanding of server managment and network security.

I mainly host game servers such as Minecraft, Titanfall 2, Satisfactory, and Terraria. I have also been working on setting up a local cloud storage on the server.
