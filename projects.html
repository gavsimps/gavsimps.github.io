Pediatric Therapy Solutions

Pediatric Therapy Solutions (PTS) is a joint company between Tiny Tots Speech Therapy and PT for Kids, Inc. to encompass physical, speech, and occupational therapy in a single setting. They had reached out to a small group of us graduates from IU to recreate their website from scratch.

We used the Flask framework to build the base of this web project. It was written using Python, HTML, CSS, Javascript, and SQL. The site allows for employee specific logins, resume management, client screenings, and automated emailing. This site is still under development, but currently fulfills all needed functions.

Web Design Final Project

This website is the culmination of the skills I learned throughout my Web Develpoment course at IU. I had a llot of fun making this project as there were no constraints on what you could make.

Ubuntu Server

I have successfully deployed and configured an external Ubuntu server as a personal project to deepen my understanding of Linux systems and server administration. I have installed and managed core servicecs such as OpenSSH, Apache/Nginx, and UFW for firewall management.

Additionally, using systemd for service management and shell scripts to automate routine tasks, this project helped in my understanding of server managment and network security.
