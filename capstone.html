Capstone 2024: BloomingLot

For my Senior Capstone project, my team and I created Bloominglot, a parking app designed to create an efficient resource for students, faculty, and visitors to find parking.

Google Map API

My primary task within the team was to set up the Google Map API and complete the functionality needed for the site. The map contains all of the IU parking lots on campus and contains information on permits types and directions.

It was so much more difficult than anticipated. I scoured over Google documentation, Stack Overflow threads, and CodePen demos just to learn how the map worked.

There is a lot of information that is not obvious from the Google maps documentation

SQL and JSON

To start the process, I set up a Google cloud project to initialize the maps API key. From there, it was just following the guide in the documentation.

The next step was getting the data needed, so I found every IU campus parking lot and added it to an SQL database with the following data:

Address Location (Latitude, Longitude) Electric Vehicle Charging
Name Permits Accessibility Parking

Then, I created a connection to the databse and sent the data out as a JSON file which I had PHP and Javascript parse.
